## cloudfuse secure encrypt

Encrypt your config file

### Synopsis

Encrypt a YAML configuration file using AES encryption.
The output file will have a .aes extension.

```
cloudfuse secure encrypt [flags]
```

  cloudfuse secure encrypt -c config.yaml -p SECRET

  # Encrypt to a specific output file
