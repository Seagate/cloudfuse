## cloudfuse unmount

Unmount container

### Synopsis

Unmount a cloudfuse mount point. Supports wildcards to unmount multiple mounts.

```
cloudfuse unmount <mount path> [flags]
```

### Examples

```
  # Unmount a specific mount point
  cloudfuse unmount ~/mycontainer

  # Lazy unmount (Linux only)
  cloudfuse unmount ~/mycontainer --lazy

  # Unmount all mounts matching a pattern
  cloudfuse unmount "~/container*"
```
